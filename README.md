# Rom::Sharding

Experimental sharding support for [Ruby Object Mapper](http://rom-rb.org/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rom-sharding', github: 'sagmor/rom-sharding'
```

And then execute:

    $ bundle
